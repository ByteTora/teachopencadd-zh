## 本教程的目标

在本教程中，我们将讨论激酶：它们为什么在生命和药物设计中很重要，它们长什么样，以及有哪些数据资源可用？
最后，我们将选择一组激酶，在接下来的教程**T024**-**T028**中分析它们的相似性，目标是深入了解潜在的脱靶效应。
