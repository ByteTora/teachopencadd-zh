# T003 · 分子筛选：不想要的子结构

## 教程目标

在我们的筛选库中，有些子结构是我们不希望包含的。在这个教程中，我们将学习到不同类型的不想要的子结构，以及如何使用RDKit来查找、突出显示和移除它们。

### 理论内容

* 不想要的子结构
* 泛分析干扰化合物（PAINS）

### 实践内容

* 加载和可视化数据
* 筛选PAINS
* 筛选不想要的子结构
* 突出显示子结构
* 子结构统计
