 ## 这次演讲的目的

在本笔记本中，您将以程序方式使用 Python 中的在线 Web 服务来学习如何在药物设计的上文中。在这次演讲结束时，您将使用REST 服务和 Web Scraping。

# 理论内容

* 从服务器端角度访问数据

 # 实用内容

* 下载静态文件 * 访问动态生成的内容 * 编程接口 * 文档解析 * 浏览器远程控制
