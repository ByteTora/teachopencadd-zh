## 本次讲座的目的


<!-- TODO: The wording of this paragraph is confusing -->

相似的化合物可能与相同的靶标结合并显示出相似的效果。
基于这种相似性质原则，化合物相似性可用于通过聚类构建化学基团。
从这样的聚类中，还可以从更大的筛选化合物集中选择不同的化合物集，以进行进一步的实验测试。


### _Theory_ 中的内容

* 聚类和 Jarvis-Patrick 算法简介
* Butina 聚类的详细说明
* 挑选多样化的化合物

### _实用_ 中的内容

* 使用 Butina 算法进行聚类
* 可视化集群
* 选择最终的化合物列表
* 奖励：运行时间分析
