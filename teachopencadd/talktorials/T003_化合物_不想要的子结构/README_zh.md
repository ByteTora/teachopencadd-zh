# T003 ·子选择：不想要的子结构

## 课程目标

在我们的筛选库中，有些子结构是我们不希望包含的。在这个课程中，我们将学习不同类型的不想要的结构，以及如何使用RDKit来查找、突出显示和删除它们。

# 理论内容

* 不想要的子结构 * 广泛分析干燥化学物质（PAIN）

# 实践内容

* 加载和可视化数据 * 选择痛苦 * 筛选不想要的子结构 * 突出显示子结构 * 子结构统计
