 ## 本期脱口秀的目标

在本笔记本中，您将学习如何在药物设计的背景下以编程方式使用Python的在线网络服务。在本期谈话结束时，您将熟悉REST服务和Web抓取。

 # 理论内容

* 从服务器端角度进行数据访问

 # 实用内容

* 下载静态文件 * 删除动态生成的内容 * 编程接口 * 文档解析 * 浏览器遥控器
