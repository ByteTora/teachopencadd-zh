    本次讲座的目的¶

    理论¶

    要成功应用 ML，我们需要一个大型分子数据集、一个分子编码、数据集中每个分子的标签，以及一个 ML 算法来训练模型。然后，我们可以对新分子进行预测。

    图 1： 机器学习概述： 分子编码、标签、ML 算法、预测。图 Andrea Volkamer 提供。

    数据准备：分子编码¶

    对于 ML，需要将分子转换为特征列表。通常使用分子指纹作为表示。

    本演讲中使用的指纹在 RDKit 中实现（更多信息可以在演讲人：G. Landrum） 是：

    • MACCS ： ‘MACCS 密钥是 166 位结构密钥描述符，其中每个位都与一个 SMARTS 模式相关联。’ （参见 OpenEye 的MACCS文档)

    • 摩根指纹 （以及 ECFP ）：’扩展连接指纹 （ECFP） 是圆形拓扑指纹，设计用于分子表征、相似性搜索和结构-活性建模。（参见 ChemAxon 的ECFP文档） ECFP 的原始实现是在 Pipeline Pilot 中完成的，它不是开源的。相反，我们使用 RDKit 中的实现，称为 Morgan 指纹。这些指纹的两个最重要的参数是半径和指纹长度。第一个参数指定每个原子考虑的圆形邻域的半径。这里考虑了两个半径：2 和 3。length 参数指定位串表示形式被哈希处理到的长度。默认长度为 2048。

    机器学习 （ML）¶

    ML 可以用来（文本改编自scikit-learn 页面):

    • 分类 （监督） ：确定对象属于哪个类别（例如：最近邻、朴素贝叶斯、RF、SVM 等）

    • 回归：预测与对象关联的连续值属性

    • 聚类（无监督）：自动将相似对象分组到集合中（另请参阅 谈话 T005 )

    监督式学习¶

    学习算法通过在训练数据中查找模式来创建规则。

    • 随机森林 （RF） ：决策树的集成。单个决策树以最大化目标函数的方式分割输入向量的特征。在随机森林算法中，生长的树是去相关的，因为分割的特征选择是随机选择的。

    • 支持向量机 （SVM） ： SVM 可以使用所谓的内核技巧有效地执行非线性分类，将其输入隐式映射到高维特征空间。分类器基于将边际最大化作为目标函数的思想。

    • 人工神经网络 （ANN） ： 人工神经网络基于称为人工神经元的连接单元或节点的集合，这些单元或节点松散地模拟生物大脑中的神经元。每个连接，就像生物大脑中的突触一样，都可以将信号从一个人工神经元传输到另一个人工神经元。接收到信号的人工神经元可以处理它，然后向连接到它的其他人工神经元发送信号。

    图 2：具有一个隐藏层的神经网络示例。图取自维基百科.

    模型验证和评估¶

    验证策略：K 折交叉验证¶

    • 这种模型验证技术以迭代方式将数据集分为两组：

      • 训练数据集：被视为训练模型的已知数据集

      • 测试数据集：然后测试模型的未知数据集

      • 过程重复 k 次

    • 目标是测试模型预测以前从未见过的数据的能力，以标记已知的过度拟合问题并评估模型的泛化能力。

    Performance measures¶

    • 灵敏度，也称真阳性率

      • TPR = TP/(FN + TP)

      • 直觉上： 在所有实际的正样本中，有多少被预测为正样本？

    • 特异性 ，也称真阴性率

      • TNR = TN/(FP + TN)

      • 直觉上： 在所有实际的负样本中，有多少被预测为负样本？

    • 准确性 ，也称真实度

      • ACC = (TP + TN)/(TP + TN + FP + FN)

      • 直觉上： 正确预测的比例。

    • ROC 曲线，受试者工作特征曲线

      • 一个图形图，说明了我们的分类器的分类能力

      • 绘制敏感度与特异性的关系图

    • AUC，ROC 曲线下面积（Area under the ROC curve，AUC）：

      • 描述分类器将随机选择的正实例排在比负实例更高的概率

      • 介于 0 和 1 之间的值，越高越好

    数据标注¶

    我们需要将每种化合物分类为活性和非活性。因此，我们使用 pIC50 值。

    • pIC50 = -log10（IC50）

    • IC50 描述了抑制所需的物质量，体外，一个进程减少 50% 。

    • 离散化 pIC50 数据的常见临界值为 6.3，我们将将其用于实验（参见 [J. Med. Chem.（2017）， 60， 474-485]（https://doi.org/10.1021/acs.jmedchem.6b01611） 和相应的 [笔记本]（https://github.com/Team-SKI/Publications/blob/master/Profiling_prediction_of_kinase_inhibitors/Build_ABL1_model.ipynb））

    • 请注意，文献中还有其他几个关于活性截止值的建议，范围从 pIC50 值 5 到 7，甚至定义何时不获取数据点的排除范围。

    def smiles_to_fp(smiles, method="maccs", n_bits=2048):
    """
将 SMILES 字符串中的分子编码为指纹。
    Parameters
    ----------
    smiles : str
        定义分子的 SMILES 字符串。

    method : str
        要使用的指纹类型。默认为 MACCS 密钥。

    n_bits : int
        指纹的长度。

    Returns
    -------
    array
        指纹数组。

    """

    # 将 smiles 转换为 RDKit mol 对象
    mol = Chem.MolFromSmiles(smiles)

    if method == "maccs":
        return np.array(MACCSkeys.GenMACCSKeys(mol))
    if method == "morgan2":
        fpg = rdFingerprintGenerator.GetMorganGenerator(radius=2, fpSize=n_bits)
        return np.array(fpg.GetFingerprint(mol))
    if method == "morgan3":
        fpg = rdFingerprintGenerator.GetMorganGenerator(radius=3, fpSize=n_bits)
        return np.array(fpg.GetFingerprint(mol))
    else:
        # NBVAL_CHECK_OUTPUT
        print(f"Warning: Wrong method specified: {method}. Default will be used instead.")
        return np.array(MACCSkeys.GenMACCSKeys(mol))

    机器学习 （ML）¶

    在下文中，我们将尝试几种 ML 方法来对我们的分子进行分类。我们将使用：

    • 随机森林 （RF）

    • 支持向量机 （SVM）

    • 人工神经网络 （ANN）

    此外，我们将对结果发表评论。

    目标是测试模型预测以前从未见过的数据的能力，以标记已知的过拟合问题并评估模型的泛化能力。

    我们首先定义一个函数model_training_and_validation它使模型在数据的随机训练-测试拆分上拟合，并返回在测试集上评估的准确性、灵敏度、特异性和 AUC 等度量值。我们还使用plot_roc_curves_for_models.

    然后，我们定义一个名为crossvalidation它执行交叉验证过程并打印 folds 上结果的统计信息。

    def plot_roc_curves_for_models(models, test_x, test_y, save_png=False):
    """
    用于绘制自定义 roc 曲线的辅助函数。

    Parameters
    ----------
    models: dict
        预训练机器学习模型的字典。
    test_x: list
        测试集的分子指纹图.
    test_y: list
        测试集的关联活动标签.
    save_png: bool
       将映像保存到磁盘（默认值 = False）

    Returns
    -------
    fig:
        Figure.
    """

    fig, ax = plt.subplots()

    # 下面的 for 循环遍历您的模型列表
    for model in models:
        # 选择模型
        ml_model = model["model"]
        # 测试集上的预测概率
        test_prob = ml_model.predict_proba(test_x)[:, 1]
        # 测试集上的 Prediction 类
        test_pred = ml_model.predict(test_x)
        # 计算假阳性率和真阳性率
        fpr, tpr, thresholds = metrics.roc_curve(test_y, test_prob)
        # 计算要在绘图上显示的曲线下的面积
        auc = roc_auc_score(test_y, test_prob)
        # 绘制计算值
        ax.plot(fpr, tpr, label=(f"{model['label']} AUC area = {auc:.2f}"))

    # 绘图的自定义设置
    ax.plot([0, 1], [0, 1], "r--")
    ax.set_xlabel("False Positive Rate")
    ax.set_ylabel("True Positive Rate")
    ax.set_title("Receiver Operating Characteristic")
    ax.legend(loc="lower right")
    # 保存图
    if save_png:
        fig.savefig(f"{DATA}/roc_auc", dpi=300, bbox_inches="tight", transparent=True)
    return fig

    def model_performance(ml_model, test_x, test_y, verbose=True):
    """
    Helper function to calculate model performance

    Parameters
    ----------
    ml_model: sklearn model object
        The machine learning model to train.
    test_x: list
        Molecular fingerprints for test set.
    test_y: list
        Associated activity labels for test set.
    verbose: bool
        Print performance measure (default = True)

    Returns
    -------
    tuple:
        Accuracy, sensitivity, specificity, auc on test set.
    """

    # 测试集上的预测概率
    test_prob = ml_model.predict_proba(test_x)[:, 1]

    # 测试集上的 Prediction 类
    test_pred = ml_model.predict(test_x)

    # 模型在测试集上的性能
    accuracy = accuracy_score(test_y, test_pred)
    sens = recall_score(test_y, test_pred)
    spec = recall_score(test_y, test_pred, pos_label=0)
    auc = roc_auc_score(test_y, test_prob)

    if verbose:
        # 打印模型表现
        # NBVAL_CHECK_OUTPUT
        print(f"Accuracy: {accuracy:.2}")
        print(f"Sensitivity: {sens:.2f}")
        print(f"Specificity: {spec:.2f}")
        print(f"AUC: {auc:.2f}")

    return accuracy, sens, spec, auc

    Helper 函数，用于在数据的随机训练测试拆分上拟合机器学习模型并返回性能度量。

    def model_training_and_validation(ml_model, name, splits, verbose=True):
    """
    在数据的随机训练-测试拆分上拟合机器学习模型并返回性能度量。

    Parameters
    ----------
    ml_model: sklearn model object
        The machine learning model to train.
    name: str
        Name of machine learning algorithm: RF, SVM, ANN
    splits: list
        List of desciptor and label data: train_x, test_x, train_y, test_y.
    verbose: bool
        Print performance info (default = True)

    Returns
    -------
    tuple:
        Accuracy, sensitivity, specificity, auc on test set.

    """
    train_x, test_x, train_y, test_y = splits

    # Fit the model
    ml_model.fit(train_x, train_y)

    # Calculate model performance results
    accuracy, sens, spec, auc = model_performance(ml_model, test_x, test_y, verbose)

    return accuracy, sens, spec, auc

    交叉验证¶

    接下来，我们将使用三种不同的模型进行交叉验证实验。 因此，我们在交叉验证循环中为机器学习模型训练和验证定义了一个辅助函数。

    [ ]:

    def crossvalidation(ml_model, df, n_folds=5, verbose=False):
    """
    交叉验证循环中的机器学习模型训练和验证。
    Parameters
    ----------
    ml_model: sklearn model object
        The machine learning model to train.
    df: pd.DataFrame
        Data set with SMILES and their associated activity labels.
    n_folds: int, optional
        Number of folds for cross-validation.
    verbose: bool, optional
        Performance measures are printed.

    Returns
    -------
    None

    """
    t0 = time.time()
    # Shuffle the indices for the k-fold cross-validation
    kf = KFold(n_splits=n_folds, shuffle=True, random_state=SEED)

    # Results for each of the cross-validation folds
    acc_per_fold = []
    sens_per_fold = []
    spec_per_fold = []
    auc_per_fold = []

    # Loop over the folds
    for train_index, test_index in kf.split(df):
        # clone model -- we want a fresh copy per fold!
        fold_model = clone(ml_model)
        # Training

        # Convert the fingerprint and the label to a list
        train_x = df.iloc[train_index].fp.tolist()
        train_y = df.iloc[train_index].active.tolist()

        # Fit the model
        fold_model.fit(train_x, train_y)

        # Testing

        # Convert the fingerprint and the label to a list
        test_x = df.iloc[test_index].fp.tolist()
        test_y = df.iloc[test_index].active.tolist()

        # Performance for each fold
        accuracy, sens, spec, auc = model_performance(fold_model, test_x, test_y, verbose)

        # Save results
        acc_per_fold.append(accuracy)
        sens_per_fold.append(sens)
        spec_per_fold.append(spec)
        auc_per_fold.append(auc)

    # Print statistics of results
    print(
        f"Mean accuracy: {np.mean(acc_per_fold):.2f} \t"
        f"and std : {np.std(acc_per_fold):.2f} \n"
        f"Mean sensitivity: {np.mean(sens_per_fold):.2f} \t"
        f"and std : {np.std(sens_per_fold):.2f} \n"
        f"Mean specificity: {np.mean(spec_per_fold):.2f} \t"
        f"and std : {np.std(spec_per_fold):.2f} \n"
        f"Mean AUC: {np.mean(auc_per_fold):.2f} \t"
        f"and std : {np.std(auc_per_fold):.2f} \n"
        f"Time taken : {time.time() - t0:.2f}s\n"
    )

    return acc_per_fold, sens_per_fold, spec_per_fold, auc_per_fold

    讨论¶

    • 哪种模型在我们的数据集上表现最好，为什么？

      • 这三种模型在我们的数据集上都表现得（非常）好。最好的模型是随机森林和支持向量机模型，它们的平均 AUC 约为 90%。我们的神经网络显示的结果略低。

      • 随机森林和支持向量机模型性能最佳可能有多种原因。我们的数据集可能很容易通过一些简单的树状决策或径向基函数来区分 active/inactive 。因此，指纹中没有如此复杂的模式来进行这种分类。

      • ANN 性能稍差的一个原因可能是训练模型的数据太少。

      • 此外，始终建议为模型评估设置另一个外部验证集。

    • MACCS 是正确的选择吗？

      • 显然，MACCS 非常适合开始训练和验证模型，看看是否可以进行分类。

      • 但是，与其他密钥（166 位）相比，MACCS 密钥相当短（2048 位），例如 Morgan 指纹。如上一个模拟所示，拥有更长的指纹有助于学习过程。所有测试模型使用 Morgan 指纹的表现略好（参见平均 AUC 增加）。
