    理论¶

    分子相似性¶

    分子相似性是化学信息学中一个众所周知且经常使用的概念。比较分子及其属性可以在许多不同的方式中使用，并帮助我们识别具有所需属性和生物活性的新分子。

    相似属性原理（SPP）以及构效关系（SAR）方法都基于这样的假设：结构相似的分子具有相似的属性，因此具有相似的生物活性。在这种情况下，虚拟筛选遵循这一理念，在给定具有已知结合亲和力的分子集合中搜索相似的分子，并将这些分子作为潜在的新活性分子提出。

    分子描述符¶

    相似性可以通过许多不同的方式进行评估，这取决于应用场景（参见J. Med. Chem. (2014), 57, 3186-3204）：

    • 1D分子描述符：溶解度，logP，分子量，熔点

      • 全局描述符：单个值代表整个分子

      • 通常不包含足够的信息以应用于机器学习（ML）

      • 可以添加到2D指纹中，以改善ML的分子编码

    • 2D分子描述符：分子图，路径，片段，原子环境

      • 分子个体部分的详细表示

      • 包含许多特征/位点，称为指纹

      • 经常用于相似性搜索和ML

    • 3D分子描述符：形状，立体化学

      • 由于分子的灵活性，不如2D表示那样稳健（分子的“正确”构象是什么？） 生物相似性

      • 生物指纹，例如，单个位点代表不同目标的生物活性度量

      • 与分子结构无关

      • 需要实验（或预测）数据

    我们已经学习了如何在演讲教程T002中计算1D物理化学参数，例如分子量和logP。 更多关于这些描述符的信息可以在rdkit文档中找到。

    接下来，我们专注于2D分子描述符的定义。由于它们通常每个分子都是独一无二的，这些描述符也被称为指纹。

    分子描述符¶

    相似性可以根据应用的不同以多种不同的方式进行评估（参见J. Med. Chem. (2014), 57, 3186-3204)：

    • 1D 分子描述符：溶解度、logP、分子量、熔点。

      • 全局描述符：只有一个值代表整个分子

      • 通常不包含足够的信息应用于机器学习（ML）

      • 可以添加到2D指纹中，以改善ML的分子编码

    • 2D 分子描述符：分子图、路径、片段、原子环境

      • 分子各个部分的详细表示

      • 包含许多特征/位点，称为指纹

      • 经常用于相似性搜索和ML

    • 3D 分子描述符：形状、立体化学

      • 由于分子的灵活性，不如2D表示鲁棒（什么是分子的“正确”构象？） 生物相似性

      • 生物指纹，例如，单个位点代表针对不同靶标的生物活性度量

      • 与分子结构无关

      • 需要实验（或预测）数据

    我们已经在Talktorial T002中学习了如何计算1D物理化学参数，例如分子量和logP。 更多关于这些描述符的信息可以在rdkit文档中找到。

    接下来，我们专注于2D分子描述符的定义。由于它们每个分子通常具有唯一性，这些描述符也被称为指纹。

    分子指纹¶

    基于子结构的指纹¶

    分子指纹是一种计算化的分子表示方法，它将化学和分子特征以位串、位向量或数组的形式编码。每一位对应一个预定义的分子特征或环境，其中“1”代表特征存在，“0”代表特征不存在。请注意，有些实现是基于计数的，因此它们会计算特定特征出现的次数。

    设计指纹的方法有多种。 在这里，我们介绍MACCS键和Morgan指纹作为两种常用的2D指纹。 正如在rdkit文档关于指纹中所见，rdkit还提供了多种替代指纹。

    MACCS(Molecular ACCess System ) 指纹¶

    MACCS指纹，也称为MACCS结构密钥，由166个预定义的结构片段组成。每个位置查询一个特定结构片段或密钥的存在或缺失。 这些单独的密钥是由药物化学家根据经验定义的，易于使用和解释（参见rdkit文档中的MACCS密钥)。

    图 1：MACCS指纹的示意图（由Andrea Morger提供）。

    Morgan指纹和圆形指纹¶

    基于Morgan算法的这类指纹，其位对应于分子中每个原子的圆形环境。考虑的邻近键和原子的数量由半径设置。还可以定义位串的长度，长于所需长度的位串将被截取至所需长度。因此，莫根指纹不局限于一定数量的位。

    更多关于Morgan指纹的信息可以在rdkit文档中找到。扩展连通性指纹（ECFP）也是常用的指纹，它们是使用莫根算法的一个变体推导出来的，详见J. Chem. Inf. Model. (2010), 50,742-754。

    图 2：Morgan圆形指纹的示意图（图由Andrea Morger提供）。

    分子相似性度量¶

    一旦计算出描述符/指纹，就可以比较它们以评估两个分子之间的相似性。分子相似性可以通过多种不同的相似性系数来量化，两种常见的相似性度量是Tanimoto和Dice指数（J. Med. Chem. (2014), 57, 3186-3204）。

    Tanimoto 系数¶

    \[T _{c}(A,B) = \frac{c}{a+b-c}\]

    • a: 分子 A 中存在的特性数量

    • b: 分子 B 中存在的特性数量

    • c: 分子 A 和 B 共有的特性数量

    Dice系数¶

    \[D_{c}(A,B) = \frac{c}{\frac{1}{2}(a+b)}\]

    • a: 分子 A 中存在的特性数量

    • b: 分子 B 中存在的特性数量

    • c: 分子 A 和 B 共有的特性数量

    相似性度量通常考虑在任一指纹中存在的正位（1’s）数量以及两者共有的正位数量。 Dice相似性通常返回比Taminoto相似性更高的值，因为它们的分母不同：

    \[\frac{c}{a+b-c} \leq \frac{c}{\frac{1}{2}(a+b)}\]

    虚拟筛选¶

    药物发现早期阶段的挑战是从庞大的现有化学空间中筛选出可能与研究目标结合的小分子集合。请注意，这个化学空间非常庞大：小分子可以由1020种化学基团的组合构成（《ACS Chem. Neurosci.》(2012), 卷19, 页649-57）。

    由于对所有这些小分子进行实验高通量筛选（HTS）以检测其对感兴趣目标的活性既耗时又成本高昂，因此采用计算机辅助方法提出一个经过筛选的小分子列表以供测试。这个过程称为虚拟（高通量）筛选：通过规则和/或模式筛选一个大的小分子库，以识别那些最有可能与研究目标结合的小分子。

    使用相似性搜索的虚拟筛选¶

    将一组新分子与一个（或多个）已知活性分子进行比较，以找到最相似的分子，可以作为一种简单的虚拟筛选方法。 根据相似性质原理，我们可以假设最相似的分子，例如与已知抑制剂相似的分子，也具有相似的效果。相似性搜索的要求如下（如上文详细讨论）：

    • 一种编码化学/分子特征的表示方法

    • 特征的潜在加权（可选）

    • 一种相似性度量

    通过计算一个分子与特定数据集中所有分子之间的相似性，可以执行相似性搜索。按相似性系数对数据集中的分子进行排名，可以得到最相似的分子排在最前面。

    富集图¶

    富集图用于验证虚拟筛选结果，它显示了在排名列表前x%中检测到的活性分子的比例，即：

    • 来自整个数据集的排名靠前的分子的比例（x轴）与

    • 来自整个数据集的活性分子的比例（y轴）

    图3：虚拟筛选结果的富集图示例

    实践¶

    在本实践部分的第一部分，我们将使用 rdkit 对分子进行编码（分子指纹）并进行比较，以计算它们的相似性（分子相似性度量），如上理论部分所讨论的。

    在第二部分，我们将使用这些编码和比较技术进行相似性搜索（虚拟筛选）： 我们使用已知的 EGFR 抑制剂 Gefitinib 作为查询，并在我们的 EGFR 测试分子数据集中搜索相似分子，这些分子是从 ChEMBL 数据库中收集的，在 Talktorial T001 中进行了收集，并在 Talktorial T002 中根据 Lipinski 的五规则进行了筛选。

    def get_enrichment_data(molecules, similarity_measure, pic50_cutoff):
    """
    Calculates x and y values for enrichment plot:
        x - % ranked dataset
        y - % true actives identified

    Parameters
    ----------
    molecules : pandas.DataFrame
        Molecules with similarity values to a query molecule.
    similarity_measure : str
        Column name which will be used to sort the DataFrame．
    pic50_cutoff : float
        pIC50 cutoff value used to discriminate active and inactive molecules.

    Returns
    -------
    pandas.DataFrame
        Enrichment data: Percentage of ranked dataset by similarity vs. percentage of identified true actives.
    """

    # Get number of molecules in data set
    molecules_all = len(molecules)

    # Get number of active molecules in data set
    actives_all = sum(molecules["pIC50"] >= pic50_cutoff)

    # Initialize a list that will hold the counter for actives and molecules while iterating through our dataset
    actives_counter_list = []

    # Initialize counter for actives
    actives_counter = 0

    # Note: Data must be ranked for enrichment plots:
    # Sort molecules by selected similarity measure
    molecules.sort_values([similarity_measure], ascending=False, inplace=True)

    # Iterate over the ranked dataset and check each molecule if active (by checking bioactivity)
    for value in molecules["pIC50"]:
        if value >= pic50_cutoff:
            actives_counter += 1
        actives_counter_list.append(actives_counter)

    # Transform number of molecules into % ranked dataset
    molecules_percentage_list = [i / molecules_all for i in range(1, molecules_all + 1)]

    # Transform number of actives into % true actives identified
    actives_percentage_list = [i / actives_all for i in actives_counter_list]

    # Generate DataFrame with x and y values as well as label
    enrichment = pd.DataFrame(
        {
            "% ranked dataset": molecules_percentage_list,
            "% true actives identified": actives_percentage_list,
        }
    )

    return enrichment

    Calculate enrichment factors¶

    The (experimental) enrichment factor (EF) is used to access a methods success by a single number rather than a plot. The idea is to determine the percentage of active molecules in the top x% (e.g. 5%) of ranked molecules, as defined by the following helper function.

    [47]:

    def calculate_enrichment_factor(enrichment, ranked_dataset_percentage_cutoff):
    """
    Get the experimental enrichment factor for a given percentage of the ranked dataset.

    Parameters
    ----------
    enrichment : pd.DataFrame
        Enrichment data: Percentage of ranked dataset by similarity vs. percentage of
        identified true actives.
    ranked_dataset_percentage_cutoff : float or int
        Percentage of ranked dataset to be included in enrichment factor calculation.

    Returns
    -------
    float
        Experimental enrichment factor.
    """

    # Keep only molecules that meet the cutoff
    enrichment = enrichment[
        enrichment["% ranked dataset"] <= ranked_dataset_percentage_cutoff / 100
    ]
    # Get highest percentage of actives and the corresponding percentage of actives
    highest_enrichment = enrichment.iloc[-1]
    enrichment_factor = round(100 * float(highest_enrichment["% true actives identified"]), 1)
    return enrichment_factor

    这个辅助函数计算在排名靠前的x%分子中活性分子的随机富集度（随机富集因子）。

    def calculate_enrichment_factor_optimal(molecules, ranked_dataset_percentage_cutoff, pic50_cutoff):
    """
    Get the optimal random enrichment factor for a given percentage of the ranked dataset.

    Parameters
    ----------
    molecules : pandas.DataFrame
        the DataFrame with all the molecules and pIC50.
    ranked_dataset_percentage_cutoff : float or int
        Percentage of ranked dataset to be included in enrichment factor calculation.
    activity_cutoff: float
        pIC50 cutoff value used to discriminate active and inactive molecules

    Returns
    -------
    float
        Optimal enrichment factor.
    """

    ratio = sum(molecules["pIC50"] >= pic50_cutoff) / len(molecules) * 100
    if ranked_dataset_percentage_cutoff <= ratio:
        enrichment_factor_optimal = round(100 / ratio * ranked_dataset_percentage_cutoff, 1)
    else:
        enrichment_factor_optimal = 100.0
    return enrichment_factor_optimal

    讨论¶

    我们学习了如何使用指纹和相似性度量来比较查询分子与分子数据集，并按相似性对分子进行排名。这种方法是虚拟筛选分子的一个例子。 富集图和富集因子可以用来评估性能。

    使用分子指纹进行相似性搜索的一个缺点是它基于分子相似性，因此不会得到任何新颖的结构。另一个挑战是在使用分子相似性时所谓的活性悬崖，即分子的一个官能团的微小变化可能会引发生物活性的跳跃。
